# Task Specs

This folder stores long prompts as durable task specs.

Use this when a prompt is too long to keep in chat context.

Process:
1. Create a new file named `YYYY-MM-DD_task-name.md`.
2. Start from `_TEMPLATE.md`.
3. Keep the spec scoped to one task.
4. Add the task to `docs/TASK_INDEX.md`.
5. After completion, update `docs/TASK_HISTORY.md`.

Only read the current task spec during execution.
